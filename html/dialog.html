Գլխավոր Html-ի տեղեկատու <dialog>

<dialog>

<dialog> տարրը ստեղծում է երկխոսության տուփ, որում կարող եք ցուցադրել հաղորդագրություն կամ ձև, օրինակ, կայք մուտք գործելու համար:

Երկխոսության վանդակը ցուցադրվում է հետևյալ նախադրված ոճով: 

position: absolute;
left: 0;
right: 0;
margin: auto;
border: solid;
padding: 1em;
background: white;
color: black;

Այսպիսով, երկխոսության տուփը ցուցադրվում է սպիտակ ֆոնով, սեւ եզրագծով և կենտրոնացած է հորիզոնական առանցքի վրա: Լայնությունը նույնն է, ինչ մայր կոնտեյների լայնությունը:

show() և close() մեթոդներն օգտագործվում են համապատասխանաբար երկխոսության դիալոգը ցույց տալու և թաքցնելու համար, ինչպես ցույց է տրված օրինակում: Այլընտրանքորեն, դուք կարող եք վերածել երկխոսությունը մոդալի ՝ օգտագործելով show() մեթոդը showModal()-ի փոխարեն: Այս դեպքում էջի մնացած տարրերն արգելափակված են. Տեքստը հնարավոր չէ ընտրել, և կոճակները չեն կարող սեղմվել, մինչև փակվի երկխոսության դիալոգը: Բացի այդ, մոդալ պատուհանը կարող է փակվել Esc ստեղնով։

Պարտադիր է:

Հատկանիշներ

open
Ցուցադրում է երկխոսության տուփ: Առանց այս ատրիբուտի ոճերում նրան ավելանում է display: none եւ պատուհանը չեն ցուցադրվում բրաուզերում։

Նաև այս տարրի համար հասանելի են հատուկ ատրիբուտներ և իրադարձություններ։

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>dialog</title> <style> body { background: url(/example/image/shark.jpg) no-repeat; background-size: cover; } dialog { background: rgba(255, 255, 255, 0.7); width: 300px; box-shadow: 0 0 5px rgba(0, 0, 0, 0.5); border-radius: 5px; } </style> </head> <body> <button id="openDialog">Открыть окно</button> <dialog> <p>Полинезийцы называют Млечный путь Манго-Роа-И-Ата, что в переводе с маори означает «Длинная акула на рассвете».</p> <p><button id="closeDialog">Закрыть окно</button></p> </dialog> <script> var dialog = document.querySelector('dialog'); document.querySelector('#openDialog').onclick = function() { dialog.show(); // Показываем диалоговое окно } document.querySelector('#closeDialog').onclick = function() { dialog.close(); // Прячем диалоговое окно } </script> </body> </html>

Այս օրինակի արդյունքը ներկայացված է նկար 1-ում: Սեղմելով «Открыть окно» оկոճակը ցուցադրում է <dialog> տարրի բովանդակությունը, որոնք նախկինում անտեսանելի էին: Սեղմելով «Закрыть окно» կոճակը, տարրի բովանդակությունը, դառնում է անտեսանելի:

Հստակեցում

Յուրաքանչյուր հստակեցում անցնում է մի քանի փուլեր հաստատման համար։

  • Recommendation (Рекомендация) — Հստակեցումը հաստատվել է W3C-ով, եւ խորհուրդ է տրվում որպես ստանդարտ:
  • Candidate Recommendation (Возможная рекомендация) — Ստանդարտի համար պատասխանատու խումբը բավարարված է, քանի որ այն համապատասխանում է իր նպատակներին, սակայն պահանջվում է մշակողների խմբի աջակցությունը ստանդարտի իրականացման համար:
  • Proposed Recommendation (Предлагаемая рекомендация) — Այս փուլում փաստաթուղթը ներկայացվել է W3C խորհրդատվական խորհրդի քննարկմանը ՝ վերջնական հաստատման համար ։
  • Working Draft (Рабочий проект) —Նախագծի ավելի հասուն տարբերակը քննարկումից հետո եւ ուղղումներ ներդրումը որպեսզի խումբը դիտարկի։
  • Editor's draft (Редакторский черновик) — Նախագծի խմբագիրների կողմից խմբագրումից հետո ստանդարտի նախագծի նախագիծ:
  • Draft (Черновик спецификации) — Ստանդարտի առաջին նախագիծը:

Առանձնատունը HTML-ի (Living) — կենդանի ստանդարտն է, այն չի պահպանում տարբերակների ավանդական համարակալումը, քանի որ գտնվում է մշտական մշակման մեջ եւ պարբերաբար թարմացվում է:
