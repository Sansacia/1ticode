Գլխավոր Html-ի տեղեկատու <script>

<script>

Элемент <script> (от англ. script — сценарий) предназначен для описания скриптов, может содержать ссылку на программу или её текст на определённом языке. Скрипты могут располагаться во внешнем файле и связываться с любым HTML-документом. Такой подход позволяет использовать одни и те же общие функции на многих веб-страницах и ускоряет их загрузку, т. к. внешний файл кэшируется при первой загрузке, и скрипт вызывается быстрее при последующих вызовах.

<script> может располагаться в заголовке или теле HTML-документа в неограниченном количестве. В большинстве случаев местоположение скрипта никак не сказывается на работу программы. Однако скрипты, которые должны выполняться в первую очередь, обычно помещают в заголовок документа.

Օրինակ 

<script>
 <скрипт>
</script> 
<script src="<адрес>"></script>

Փակող թեգ

Պարտադիր է:

Հատկանիշներ

async
Загружает скрипт асинхронно.
defer
Откладывает выполнение скрипта до тех пор, пока вся страница не будет загружена полностью.
language
Устанавливает язык программирования на котором написан скрипт.
src
Адрес скрипта из внешнего файла для импорта в текущий документ.
type
Определяет тип содержимого <script>.

Օրինակ

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd"> <html> <head> <meta http-equiv="Content-Type" content="text/html; charset=utf-8"> <title>SCRIPT</title> </head> <body> <script type="text/javascript"> document.write ('<table width="100%" border="1">'); for (i=1; i<6; i++) { document.writeln("<tr>"); for (j=1; j<6; j++) document.write("<td>" + i + j + "<\/td>"); document.writeln("<\/tr>"); } document.write ("<\/table> "); </script> </body> </html>

Результат данного примера показан на рис. 1.

Рис. 1. Результат работы скрипта

В данном примере с помощью скрипта выводится таблица, состоящая из пяти строк и столбцов, которая заполняется числами.

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>SCRIPT</title> <script> function popup() { document.getElementById('welcome').innerHTML = 'Добро пожаловать!'; } </script> </head> <body onload="popup()"> <div id="welcome"></div> </body> </html>

В данном примере внутри элемента с идентификатором welcome выводится текст сообщения.

Примечание

В HTML5 атрибут type можно опустить, он является необязательным и принимает значение text/javascript, если не указан явно. В предыдущих версиях HTML атрибут type необходим.

Հստակեցում

Յուրաքանչյուր հստակեցում անցնում է մի քանի փուլեր հաստատման համար։

  • Recommendation (Рекомендация) — Հստակեցումը հաստատվել է W3C-ով, եւ խորհուրդ է տրվում որպես ստանդարտ:
  • Candidate Recommendation (Возможная рекомендация) — Ստանդարտի համար պատասխանատու խումբը բավարարված է, քանի որ այն համապատասխանում է իր նպատակներին, սակայն պահանջվում է մշակողների խմբի աջակցությունը ստանդարտի իրականացման համար:
  • Proposed Recommendation (Предлагаемая рекомендация) — Այս փուլում փաստաթուղթը ներկայացվել է W3C խորհրդատվական խորհրդի քննարկմանը ՝ վերջնական հաստատման համար ։
  • Working Draft (Рабочий проект) —Նախագծի ավելի հասուն տարբերակը քննարկումից հետո եւ ուղղումներ ներդրումը որպեսզի խումբը դիտարկի։
  • Editor's draft (Редакторский черновик) — Նախագծի խմբագիրների կողմից խմբագրումից հետո ստանդարտի նախագծի նախագիծ:
  • Draft (Черновик спецификации) — Ստանդարտի առաջին նախագիծը:

Առանձնատունը HTML-ի (Living) — կենդանի ստանդարտն է, այն չի պահպանում տարբերակների ավանդական համարակալումը, քանի որ գտնվում է մշտական մշակման մեջ եւ պարբերաբար թարմացվում է:
