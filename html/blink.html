Գլխավոր Html-ի տեղեկատու <blink>

<blink>

Элемент <blink> (от англ. blink — мигание, мерцание) устанавливает мигание текста.

Это нестандартный элемент, не применяйте его, взамен воспользуйтесь стилями.

Օրինակ 

<blink>Текст</blink>

Փակող թեգ

Պարտադիր է:

Հատկանիշներ

Нет.

Օրինակ

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd"> <html> <head> <meta http-equiv="Content-Type" content="text/html; charset=utf-8"> <title>BLINK</title> </head> <body> <p><blink>Внимание!</blink></p> <p>Не тыкайте в него палкой, он не мёртвый, просто прикидывается.</p> </body> </html>

Примечание

В качестве альтернативы применяется стилевое свойство animation, в параметрах которого указывается бесконечная анимация.<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>Мигание</title> <style> .blink { animation: blink 2s infinite; } @keyframes blink { from { opacity: 1; } to { opacity: 0; } } </style> </head> <body> <p class="blink">Внимание</p> </body> </html>
