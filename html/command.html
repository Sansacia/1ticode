Գլխավոր Html-ի տեղեկատու <command>

<command>

Ստեղծում է հրաման անջատիչի, վանդակի կամ սովորական կոճակի տեսքով: <Command>տարրը պետք է տեղադրված լինի <menu>-ի ներսում, հակառակ դեպքում այն ​​չի ցուցադրվի:

Սա ոչ ստանդարտ տարր է և չի աջակցվում բրաուզերի կողմից:

Օրինակ 

<menu>
 <command պարամետրերը>
</menu>

Փակող թեգ

Պարտադիր չէ:

Հատկանիշներ

checked
Ակտիվացնում է հրամանը: Հասանելի է միայն այն դեպքում, երբ type-ը տեղադրված է որպես radio կամ checkbox.
disabled
Նշում է ՝ հրամանը մատչելի է, թե ոչ:
icon
Նկարի հասցեն, որը ցուցադրվում է հրամանի դիմաց:
label
Հրամանի անվանումը։
radiogroup
Անվանում է անջատիչների խմբի անունը: Հասանելի է միայն այն դեպքում, երբ type-ը տեղադրված է որպես radio :
type
Որոշում է հրամանի տեսակը (checkbox, command, radio)։ Ըստ ստանդարտի command :

Օրինակ

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>COMMAND</title> </head> <body> <menu> <command onclick="alert('Слава роботам!')" label="Выполнить директиву №1"> <command onclick="alert('Сдавайтесь, людишки!')" label="Выполнить директиву №2"> <command onclick="alert('Убить всех человеков!')" label="Выполнить директиву №3"> </menu> </body> </html>

Բրաուզերներ

Բրաուզերների աղյուսակում կիրառվում են հետևյալ նշումները:

  •  — Տարրը ամբողջությամբ ապահովվում է զննարկչի կողմից;
  •  — Տարրը չի ընկալվում զննարկչի կողմից և անտեսվում է;
  •  — Գործողության ընթացքում կարող են հայտնվել տարբեր սխալներ, կամ էլեմենտը աջակցվում է նախազգուշացումներով;
