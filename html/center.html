Գլխավոր Html-ի տեղեկատու <center>

<center>

Элемент <center> (от англ. center — центр, середина) выравнивает содержимое контейнера по центру относительно родительского элемента.

Использование этого элемента осуждается спецификацией HTML, вместо него применяйте стили.

Օրինակ 

<center>Текст</center>

Փակող թեգ

Պարտադիր է:

Օրինակ

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd"> <html> <head> <title>CENTER</title> <meta http-equiv="Content-Type" content="text/html; charset=utf-8"> </head> <body> <center> <p>Разум — это Будда, а прекращение умозрительного мышления — это путь. Перестав мыслить понятиями и размышлять о путях существования и небытия, о душе и плоти, о пассивном и активном и о других подобных вещах, начинаешь осознавать, что разум — это Будда, что Будда — это сущность разума, и что разум подобен бесконечности.</p> </center> </body> </html>

Результат данного примера показан на рис. 1.

Рис. 1. Текст, выровненный по центру веб-страницы

Նշում

В CSS есть несколько способов выравнивания содержимого по центру. Для текста самый простой вариант — это задать для него стилевое свойство text-align со значением center.

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>center</title> <style> .center { text-align: center; } </style> </head> <body> <p class="center">Разум — это Будда, а прекращение умозрительного мышления — это путь.</p> </body> </html>
